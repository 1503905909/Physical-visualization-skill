# 高中物理电磁学主题详述

本文档详细介绍第一阶段优先覆盖的 6 个核心主题。

## 1. 库仑定律 (Coulomb's Law)

### 物理背景
库仑定律是电学的基础，描述两个点电荷之间的相互作用力。

### 核心公式
$$F = k \frac{q_1 q_2}{r^2}$$

其中：
- $F$ 是相互作用力 (N)
- $k = 8.99 \times 10^9 \, \text{N·m}^2/\text{C}^2$ 是库仑常数
- $q_1, q_2$ 是电荷量 (C)
- $r$ 是两电荷间距离 (m)

### 教学要点
- **同号相斥，异号相吸** - 定性理解
- **反平方律** - $F \propto 1/r^2$ 的理解
- **矢量性** - 力是矢量，方向沿连线
- **对称性** - $F_{12} = F_{21}$ (牛顿第三定律)

1. 绘制两个点电荷 (正负号用颜色区分)
2. 用箭头展示相互作用力方向
3. 改变距离时动态显示力的大小变化
4. 用数字标注显示 $F = k q_1 q_2 / r^2$ 的计算过程

- **显示结果**: 力的大小 (N), 用箭头显示方向
- **图表**: $F$ 与 $r$ 的关系曲线 (双对数图表现反平方律)

## 2. 电场强度 (Electric Field Strength)

### 物理背景
电场强度描述电场在空间中的强弱，是理解电现象的核心概念。

### 核心公式

定义式：
$$E = \frac{F}{q}$$

点电荷的电场强度：
$$E = k\frac{Q}{r^2}$$

其中：
- $E$ 是电场强度 (N/C 或 V/m)
- $Q$ 是场源电荷 (C)
- $r$ 是到场源的距离 (m)

### 教学要点
- **场的概念** - 电场是一种物质，客观存在
- **叠加原理** - 多个电荷的场矢量合成
- **方向定义** - 正电荷受力方向为正方向
- **电场线** - 电场线密度与场强大小成正比

### Manim 动画内容建议
1. 单点电荷的电场线辐射状图
2. 正负电荷的电场线分布
3. 矢量叠加演示 (两点电荷的合场)
4. 等量异号电荷的电场线分布

### Streamlit 交互实验建议
- **可调参数**: 电荷 $Q$ (nC), 距离 $r$ (cm)
- **显示结果**: $E$ 值 (N/C)
- **图表**: 二维电场分布热图, $E$ 与 $r$ 的曲线

## 3. 电势和电势差 (Electric Potential and Potential Difference)

### 物理背景
电势是标量概念，代表电场中单位正电荷的潜在能。电势差是不同点之间的电势之差。

### 核心公式

定义式：
$$\varphi = \frac{W}{q} = \frac{E_p}{q}$$

电势差：
$$U_{AB} = \varphi_A - \varphi_B = \frac{W}{q}$$

匀强电场中：
$$U = Ed$$

点电荷的电势：
$$\varphi = k\frac{Q}{r}$$

其中：
- $\varphi$ 是电势 (V)
- $U$ 是电势差 (V)
- $W$ 是电场力做功 (J)
- $E$ 是电场强度 (N/C)
- $d$ 是沿场方向的距离 (m)

### 教学要点
- **标量性** - 与电场强度相比是标量，没有方向
- **相对性** - 电势是相对的，有参考点
- **等势面** - 同一等势面上的点电势相同，电场线垂直于等势面
- **功能关系** - 电势反映电场对正电荷的做功能力

### Manim 动画内容建议
1. 电势与距离的关系（双曲线）
2. 单点电荷的等势面分布（同心圆）
3. 电势和电场线的关系演示
4. 正负电荷的等势面分布

### Streamlit 交互实验建议
- **可调参数**: 电荷 $Q$ (nC), 距离范围
- **显示结果**: 点的电势值 (V), 电势能 (J)
- **图表**: 等势线分布图, $\varphi$ 与 $r$ 的曲线, 三维电势分布图

## 4. 电容器 (Capacitor)

### 物理背景
电容器是一个重要的电学器件，用于存储电荷和能量。平行板电容器是最常见的类型。

### 核心公式

电容定义：
$$C = \frac{Q}{U}$$

平行板电容器：
$$C = \varepsilon_0 \varepsilon_r \frac{S}{d}$$

电容器存储的能量：
$$E_c = \frac{1}{2}CU^2 = \frac{1}{2}QU$$

其中：
- $C$ 是电容 (F)
- $Q$ 是储存的电荷量 (C)
- $U$ 是电压 (V)
- $\varepsilon_0 = 8.85 \times 10^{-12}$ F/m 是真空介电常数
- $\varepsilon_r$ 是相对介电常数
- $S$ 是极板面积 (m²)
- $d$ 是极板间距 (m)

### 教学要点
- **电容的概念** - 储存电荷的能力 (不是储存电荷的多少)
- **平行板电容器** - 理解几何参数对电容的影响
- **介质的作用** - 介质会增加电容
- **能量存储** - 电容器中的电能

### Manim 动画内容建议
1. 平行板电容器的结构演示
2. 充电过程：电荷积累、电场形成、电压建立
3. 极板间电场均匀分布的展示
4. $C$ 与 $S/d$ 的关系推导

### Streamlit 交互实验建议
- **可调参数**: 极板面积 $S$ (cm²), 间距 $d$ (mm), 电压 $U$ (V), 介质选择
- **显示结果**: 电容 $C$ (μF), 电荷 $Q$ (μC), 能量 (μJ)
- **图表**: $C$ 与 $S$ 的关系, $C$ 与 $d$ 的关系, 能量随电压的变化

## 5. 欧姆定律 (Ohm's Law)

### 物理背景
欧姆定律是电学中最基础的规律，描述电压、电流和电阻的关系。

### 核心公式

欧姆定律（导体）：
$$U = IR$$

电阻：
$$R = \rho \frac{L}{S}$$

电功率：
$$P = UI = I^2R = \frac{U^2}{R}$$

其中：
- $U$ 是电压 (V)
- $I$ 是电流 (A)
- $R$ 是电阻 (Ω)
- $\rho$ 是电阻率 (Ω·m)
- $L$ 是导体长度 (m)
- $S$ 是导体截面积 (m²)

### 教学要点
- **线性关系** - 在一定范围内 $U$ 和 $I$ 成正比
- **U-I 特性曲线** - 不同电阻的曲线斜率不同
- **电阻的定义** - $R = U/I$ 的理解
- **功率概念** - 电功率代表能量转换速率

### Manim 动画内容建议
1. 简单电路图演示
2. 欧姆定律 $U = IR$ 的推导过程
3. 不同电阻的 U-I 曲线对比
4. 电功率的三种表现形式 ($P = UI$, $P = I^2R$, $P = U^2/R$) 的演示

### Streamlit 交互实验建议
- **可调参数**: 电压 $U$ (V), 电阻 $R$ (Ω)
- **显示结果**: 电流 $I$ (A), 功率 $P$ (W)
- **图表**: U-I 曲线, 多条曲线对比不同电阻, 功率与电压关系曲线

## 6. 串并联电路 (Series and Parallel Circuits)

### 物理背景
串联和并联是两种基本的电路连接方式，具有不同的电压、电流和电阻分配规律。

### 核心公式

**串联电路**：
$$I_1 = I_2 = I = \text{总电流}$$
$$U = U_1 + U_2 + \cdots$$
$$R_{\text{总}} = R_1 + R_2 + \cdots$$

**并联电路**：
$$U_1 = U_2 = U = \text{总电压}$$
$$I = I_1 + I_2 + \cdots$$
$$\frac{1}{R_{\text{总}}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots$$

### 教学要点
- **串联分压** - 电压在电阻上分配，与电阻成正比
- **并联分流** - 电流在支路分配，与电阻成反比
- **等效电阻** - 理解电阻的串并联计算
- **实际应用** - 分压器、分流器等实际电路

### Manim 动画内容建议
1. 串联电路图与并联电路图对比
2. 电流、电压在串联中的分配演示
3. 电流、电压在并联中的分配演示
4. 等效电阻的概念和计算过程

### Streamlit 交互实验建议
- **可调参数**: 
  - 串联：$R_1$, $R_2$, $U$ (总电压)
  - 并联：$R_1$, $R_2$, $U$ (总电压)
- **显示结果**: 各支路电流、电压、功率
- **图表**: 电路图, 各量数值显示, 功率分配柱状图

## 教学建议

### 学习顺序
建议按上述顺序学习：
1. **库仑定律** → 理解电相互作用的本质
2. **电场强度** → 从点对点的作用转向场的概念
3. **电势** → 从矢量（电场）到标量（电势）
4. **电容器** → 电势的实际应用
5. **欧姆定律** → 从个体粒子到宏观电流现象
6. **电路** → 综合应用前面的概念

### 教学策略
- **由浅入深** - 从单个概念逐步构建完整体系
- **数形结合** - 既要公式，也要图像和现象
- **动静结合** - Manim 展示过程，仿真展示实时变化
- **理论实践结合** - 讲解理论后立即用仿真验证

### 课程安排示例
每个主题可以设计一个 45 分钟的课程：
- 5 分钟：导入（播放 Manim 动画）
- 15 分钟：理论讲解
- 15 分钟：Streamlit 仿真演示与互动
- 10 分钟：学生练习与反思
