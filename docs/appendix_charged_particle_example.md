## 二、物理原理

### 2.1 核心公式

粒子进入匀强电场后，受到恒定的电场力：

$$F = qE$$

根据牛顿第二定律，竖直方向加速度：

$$a = \frac{qE}{m}$$

水平方向做匀速运动（忽略重力）：

$$x = v_0 t$$

竖直方向做初速度为零的匀加速运动：

$$y = \frac{1}{2} a t^2 = \frac{1}{2} \cdot \frac{qE}{m} \cdot \left(\frac{x}{v_0}\right)^2$$

由此可得偏转轨迹为抛物线：

$$y = \frac{qE}{2mv_0^2} \cdot x^2$$

### 2.2 物理规律

- 偏转方向取决于电荷电性：**正电荷**受力方向与电场方向一致（向下偏转），**负电荷**受力方向与电场方向相反（向上偏转）。
- 偏转程度与电场强度 $E$ 成正比，与粒子质量 $m$ 成反比，与初速度 $v_0$ 的平方成反比。

### 5.3 课堂观察问题

仿真内置了三个引导性观察问题：

1. **改变粒子电性后，受力方向和轨迹偏转方向如何变化？**
   - 预期观察：切换为负电荷后，粒子向上偏转，受力箭头反向。
2. **增大电场强度时，偏转程度为什么会增大？**
   - 预期观察：$E$ 增大 → $F = qE$ 增大 → $a = F/m$ 增大 → 偏转量 $y$ 增大。
3. **增大初速度时，粒子在电场中的运动时间怎样变化？偏转怎样变化？**
   - 预期观察：$v_0$ 增大 → 通过电场时间 $t = L/v_0$ 变短 → 偏转量 $y$ 减小。

## 九、技术实现要点

### 9.1 核心渲染逻辑

```javascript
// 计算偏转量
function deflectionAt(t) {
    const T = 1 / state.v0;
    return 0.5 * acceleration() * Math.pow(t * T, 2);
}

// 计算粒子轨迹点
function pathPoint(t) {
    const x = left + t * (right - left);            // 水平匀速
    const y = midY + deflectionAt(t) * yScale;       // 竖直匀加速
    return { x, y };
}
```

### 9.2 关键技术

- **Canvas 2D**：使用 HTML5 Canvas 实现粒子轨迹的实时绘制
- **requestAnimationFrame**：使用浏览器动画帧驱动粒子运动，帧率约 60fps
- **响应式布局**：使用 CSS Grid + 媒体查询适配桌面和移动端
- **独立单文件**：所有 HTML/CSS/JavaScript 内聚于一个文件，无需外部依赖
